Exercise 00
Exercise 00.


- Does the Configuration class contain a single constructor that accepts a collection of elements of the IConfigurationSource type?
- Does IConfigurationSource have a method that returns a collection of parameters?
- Is the Params property a hash table? Is the setting of Params implemented in the class constructor?

Exercise 01
Exercise 01.


- Does the JsonSource class inherit the IConfigurationSource interface and does it implement a method that returns a collection of parameters?
- Does the JsonSource class contain a single constructor that accepts the path to the json file, where the configuration will be loaded from?
- The program should read data from the attached files, they should not be copied directly into the code. Is it so?
- Is it output to the console: Configuration Port: 1234 CheckForUpdates: True Domain: http://localhost Source: JSON

Exercise 03.

Exercise 04.


- Do the IConfigurationSource interface and its implementations have a property that reflects the loading priority?
