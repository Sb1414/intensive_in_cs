Exercise 00
Exercise 00 - How much money does money cost?


- The program should read the exchange rates from files at the specified path, they should not be copied directly into the code. Is it so?
- The application should read data from files. The path to the folder with these files should be an input argument. Is it so?
- Is information about the currency stored in the struct?
- Does the program issue a validation error in case of insufficient or incorrect input parameters?
- Does the program give the correct answer for the following input data? $ dotnet run “100 RUB” “path/to/file” Amount in the original currency: 100.00 RUB Amount in USD: 1.36 USD Amount in EUR: 1.11 EUR
- Does the program give the correct answer for the following input data? $ dotnet run “100 EUR” “path/to/file” Amount in the original currency: 100.00 EUR Amount in USD: 81.97 USD Amount in EUR: 8,990.38 RUB
- Does the program give the correct answer for the following input data? $ dotnet run “100 USD” “path/to/file” Amount in the original currency: 100.00 USD Amount in USD: 7,369.74 RUB Amount in EUR: 121.99 EUR
